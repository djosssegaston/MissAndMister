## Backend local

Pour eviter les erreurs d'upload video dues a la limite PHP CLI par defaut (`2M`), demarrez le backend Laravel avec:

```bash
cd backend/laravel
composer serve
```

Pour la stack de developpement complete:

```bash
cd backend/laravel
composer dev
```

Details: `backend/laravel/docs/upload-limits.md`
